# Sistema de Produção

Este projeto é um sistema simples desenvolvido para realizar o controle de produção de uma empresa, permitindo registrar e acompanhar informações relacionadas ao processo produtivo.

## 📌 Descrição

O sistema foi criado com o objetivo de facilitar o controle das atividades de produção, permitindo organizar e visualizar os dados de forma mais prática.

A aplicação permite registrar informações e acompanhar o andamento do processo produtivo através de uma interface web.

## 🛠 Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS
- SQLite

## ⚙️ Como executar o projeto

1. Clonar o repositório:
git clone https://github.com/guilhermefernandes20/sistema-producao.git


2. Entrar na pasta do projeto

3. Executar o arquivo principal:


python app.py


4. Abrir no navegador:


http://localhost:5000


## 🎯 Funcionalidades

- Registro de informações de produção  
- Controle e visualização dos dados  
- Interface web simples para utilização
